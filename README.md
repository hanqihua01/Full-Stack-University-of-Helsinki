# Full Stack - University of Helsinki

This is my learning record of the Full Stack open course published by the University of Helsinki. [Here](https://fullstackopen.com/) is the link to the course website.

## Quick Start

To run the react application, here are some necessary steps to do.

Let's take the ./part1/practice1.6-1.11 as an example.

1. Open the ./part1/practice directory.
2. Run `npm install` in the terminal.
3. Run `npm start` in the terminal.
